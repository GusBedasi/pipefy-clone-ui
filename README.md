# Clone UI Pipefy

Objetivo em clonar a interface do Pipefy é desenvolver habilidades de estilização com CSS usando o styled-components e adquirir novos conhecimentos passados pelo ministrador da aula Diego Fernandes.

# Usage
Basta clonar o repositório e baixar as dependências do package.json, para fazer tudo isso basta rodar os comandos abaixo:
```
git clone https://github.com/GusBedasi/pipefy-clone-ui.git
yarn add
yarn start
```

# Challenge

Fazer a criação desta interface sozinho após alguns dias desenvolendo ela junto a vídeo aula e estudando cada uma das features criadas pelo Diego3g
